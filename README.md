# Homebrew Tap

This is a custom Homebrew tap for distributing formulas.

## Installation

```bash
brew tap satococoa/tap
```

## Available Formulas

- `wtp`
- `wtcp`
- `git-worktreeinclude`

## Usage

After tapping, you can install formulas with:

```bash
brew install <formula-name>
```
